🔍 Verify Your Download

Before installing VaultMorph, verify the authenticity of your downloaded file using the official checksums below:

File Name: VaultMorphAIShield_Setup_v1.0.0.exe
File Size: [62.85 MB]
SHA256 Hash: [E874B315114B6DD0CD6501BC6DCF2776245D359EE1707F793E5E2E137279B2C6]
Certificate Thumbprint: [ED569AE15FE2290B01465695B0F09DE3EA8925BF]

⚠️ Important: Self-Signed Certificate Notice

VaultMorph uses a self-signed certificate. This means Windows will show an "Unknown Publisher" warning during installation. This is normal and does not mean the software is unsafe.

We provide this verification page so you can independently confirm the file's authenticity before installing.

How to Verify

1 Windows Command Prompt (Easy)

  • Open Command Prompt (search "cmd" in Start menu)
  • Navigate to your Downloads folder
  • Run this command:
certutil -hashfile VaultMorphAIShield_Setup_v1.0.0.exe SHA256
  • Compare the output with the SHA256 hash shown above
  • If they match, your download is authentic ✅

2 PowerShell Method

  • Right-click Start menu → Windows PowerShell
  • Navigate to your Downloads folder
  • Run this command:
Get-FileHash -Path "VaultMorphAIShield_Setup_v1.0.0.exe" -Algorithm SHA256
  • Compare the Hash value with the SHA256 shown above

3 Digital Signature Check (GUI)

  • Right-click the downloaded file → Properties
  • Go to the "Digital Signatures" tab
  • Select the signature → Click "Details"
  • Click "View Certificate"
  • Verify the certificate details match:
    • Subject: CN=VaultMorph
    • Thumbprint matches the value shown above

4 Download Verification Script

For automatic verification, download our verification script:

Download verify_checksum.bat
  • Place verify_checksum.bat in the same folder as the installer
  • Run the script by double-clicking it
  • It will automatically verify the checksum and show the result
