# Security Policy

## Supported Versions

| Version | Supported |
|--------|----------|
| 1.0.x  | Yes      |
| <1.0   | No       |

---

## Reporting a Vulnerability

VaultMorph takes security seriously.

If you discover a vulnerability, please report it responsibly.

DO NOT create a public GitHub issue.

Instead, contact:

Email: contact@vaultmorph.org

Please include:
- Description of the vulnerability
- Steps to reproduce
- Possible impact
- Suggested fix (if any)

---

## Response Timeline

- Initial response: within 48 days
- Status update: within 180 days
- Fix timeline: based on severity

---

## Responsible Disclosure

Please do not publicly disclose vulnerabilities until they have been fixed.

We will notify you once a patch is available.

---

## Scope

This policy applies to:
- VaultMorph application
- Encryption mechanisms
- Local storage and vault system

Out of scope:
- Issues caused by user negligence
- Compromised user devices
- Third-party libraries
